SLIDING PUZZLE

How To Play:

Please drag or click one of the image tiles to the empty tile square. Click the scramble button to restart the game. Click the SOLVE button to let the program solve current game and then click PLAY button to see how it solved step by step.

Please Note: the solution is not the shortest path solution.

Correct image

Algorithm:

Used modified A* Search Algorithm to find the correct path from current state to goal state, used Manhattan Distance Algorithm as the Heuristic function.

Tried with both A* and IDA* Search Algorithm to find the shortest path. Due to the running time issue, I decided to use non-shortest time solution. It takes a relatively long time to run JavaScript in DOM to find the shortest path. IDA* is faster than A*, but still running too long.(> mins). I compared with the same code written in Java, running time is significantly different (a couple seconds versus a couple minutes).

Please check Java, Ruby and JavaScript code in the package.
